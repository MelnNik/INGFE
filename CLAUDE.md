# Claude Instructions

This repository contains Claude-compatible skill folders under `skills/` and slash-command wrappers under `commands/`.

After installation, use:

- `/ingfe-plan` for project planning, docs creation, PRD/architecture alignment, and milestone planning.
- `/ingfe-execute` for implementing the earliest incomplete `PLAN.md` milestone.

If slash commands are unavailable, use the native Skill tool when available:

- Load `ingfe-plan` for project planning, docs creation, PRD/architecture alignment, and milestone planning.
- Load `ingfe-execute` for implementing the earliest incomplete `PLAN.md` milestone.

For repository maintenance, follow `AGENTS.md`.
